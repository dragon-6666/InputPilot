# README 图片素材

这里放 GitHub README 用的截图图片。

## 推荐格式

- 格式：`png`
- 宽度：建议 `1200px` 或 `1600px`
- 命名：英文小写 + 中横线
- 背景：尽量使用浅色模式截图，窗口边缘保留一点留白
- 隐私：截图前不要展示个人账号、路径、输入内容、私有应用名

## 推荐截图

建议放 4 张：

1. `settings-basic.png`
   - 基础设置页面
   - 展示全局输入法、输入法守护、开机启动

2. `settings-rules.png`
   - 应用规则页面
   - 展示针对单个 App 配置输入法

3. `floating-focus.png`
   - 焦点跟随悬浮提示
   - 展示输入框光标附近的小输入法 icon

4. `menubar.png`
   - 菜单栏状态菜单
   - 展示菜单栏 icon、当前输入法、启停守护、打开设置

## 可选截图

5. `settings-floating.png`
   - 悬浮提示设置页面
   - 展示样式、自动消失、透明度、图标大小等配置

6. `permission.png`
   - 权限页面
   - 展示辅助功能权限说明
